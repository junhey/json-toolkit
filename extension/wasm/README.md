# Chrome Extension WASM Files

This directory is populated during the build process.

## How it works

1. The root `pnpm wasm:build` command compiles the Rust core to WASM and outputs to `web/src/wasm/`
2. Before building the extension, copy the WASM files here:
   ```bash
   pnpm wasm:build
   cp web/src/wasm/* extension/wasm/
   pnpm --filter extension build
   ```
3. The `wasm/` directory is declared in `web_accessible_resources` in `wxt.config.ts`
4. The extension loads WASM via `chrome.runtime.getURL('wasm/json_core_bg.wasm')`

## Files

- `json_core.js` - JS glue code generated by wasm-pack
- `json_core_bg.wasm` - Compiled WASM binary
- `package.json` - Package metadata (not used at runtime)
